# IMUFHE
Dead reckoning with FHE POC
## Dependencies
* openFHE
## Build
```git clone https://github.com/eaglclaws/IMUFHE```
\
```cd IMUFHE```
\
```mkdir build```
\
```cd build```
\
```cmake ..```
\
```cmake --build .```

## Run
```cp ../src/data.csv .```

You must copy the **data.csv** into the same directory of the binary.

```./IMUFHE```
